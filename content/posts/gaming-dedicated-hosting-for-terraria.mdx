Quick and clean setup for hosting a dedicated Terraria Server.

1. Install Terraria on Steam
2. Steam Library → Terraria → Right-click → Manage → Browse Local Files
3. Run `Terraria Server.exe`
- you might have to run `xnafx40_redist.msi`
4. Go through prompts

Open Terraria → Multiplayer → Join via IP

Enter 127.0.0.1 or localhost

Port: 7777 (or whatever you chose)

Enter the password (if any)
