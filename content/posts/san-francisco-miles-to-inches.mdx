Developing a 1:63,360 scale model of San Francisco (1 mile = 1 inch) using 3D printing to fabricate molds for casting quick-set cement.


## Materials

Discount Builder Supply SF

- [CRC Heavy Duty Silicone Lubricant](https://www.discountbuilderssupplysf.com/p/crc-heavy-duty-silicone-lubricant/27904435)
- [Fast-Setting Quikrete](https://www.discountbuilderssupplysf.com/p/quikrete-fast-setting-concrete-mix-50-lb/61050035)

> Concrete should have **Fine aggregate or minimal aggregate** to preserve mold detail.




## Blender Model with BLOSM

NorCal OpenStreetMapData: [norcal-latest.osm.pbf](https://download.geofabrik.de/north-america/us/california/norcal.html)

Lat Long for all of SF

## Step 3: How to use in BLOSM

* In BLOSM panel, paste coordinates into the **Bounding Box** fields.
* Import one tile at a time (terrain optional, buildings essential).
* Save each as `sf_tile_1.blend`, etc.

## Scaling

1. Select all objects (terrain + buildings).
2. `Ctrl + A` → Apply Scale/Rotation (to reset before scaling).
3. Press `S` to scale.
4. Type: 0.01578

> `Ctrl + A` → Apply Scale again, so the new dimensions are “baked in”.
