## Components

- **Cloudflare:** Provides security and caching.
- **GitHub:** Code repository for all services.
- **Jenkins:** Handles CI/CD.
- **Linux Server:** Runs all services and Jenkins.

## Services Hosted

1. **Static Site:** Served by Nginx.
2. **Static Site - Python:** Generated by a static site generator.
3. **API Endpoint - JS:** Node.js application with Express.js.
4. **API Endpoint - Python:** Flask/FastAPI application.
5. **SSR Site:** Server-side rendered site with Next.js/Nuxt.js.
6. **Backend - SQL:** Backend service with SQL database.

## CI/CD with Jenkins

### Build and Deployment Process

1. **Trigger:** Builds are triggered by GitHub webhooks.
2. **Build:** Appropriate build commands based on service type.
3. **Test:** Automated tests are run.
4. **Deploy:** Deployment to production server.

## Configuration Steps

1. Set up Cloudflare to point to your Linux server IP.
2. Configure GitHub repositories for each service.
3. Set up Jenkins with pipelines for each service type.
4. Ensure all services are properly containerized or virtualized for isolation and easy deployment.

## Diagram

```
Internet
    │
 [Cloudflare]
    │
    ▼
[Linux Server] ──> [Jenkins] ──> [GitHub]
    │
    ├──> [Nginx] ──> [Static Site]
    │
    ├──> [Nginx] ──> [Static Site - Python]
    │
    ├──> [Node.js] ─> [API Endpoint - JS]
    │
    ├──> [Python] ──> [API Endpoint - Python]
    │
    ├──> [Node.js/Nginx] ─> [SSR Site]
    │
    └──> [Backend - SQL] ─> [SQL Database]
```

1. Cloudflare: Sits at the entry point from the internet, providing security features like DDoS protection and caching for faster content delivery.
1. Linux Server: Hosts all components including Jenkins.
1. Jenkins: Automated CI/CD tool that fetches code from GitHub repositories, builds, tests, and deploys them.
1. GitHub: Source code repositories connected to Jenkins for webhook-based triggers.
1. Nginx/Node.js/Python: Servers running different types of sites and APIs.
1. SQL Database: Database backend for services requiring persistent data storage.
